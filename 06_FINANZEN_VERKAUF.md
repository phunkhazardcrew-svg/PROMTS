## [FIN-01] FAHRZEUG VERKAUFSKALKULATION

**Wann:** Auto, Motorrad oder ähnliches verkaufen (eBay Kleinanzeigen / Mobile.de).

```
Erstelle eine professionelle Verkaufskalkulation.

OBJEKT: [BESCHREIBEN: Marke, Modell, Baujahr, KM-Stand]
ZUSTAND: [ZUSTAND BESCHREIBEN: Mängel, Stärken, Ausstattung]
KAUFPREIS DAMALS: [BETRAG / unbekannt]
MEINE SCHMERZGRENZE (NUR FÜR MICH SICHTBAR): [BETRAG]
VERKAUFSPLATTFORM: [eBay Kleinanzeigen / Mobile.de / Facebook Marketplace]

RECHERCHE-AUFGABE:
- Aktuellen Neupreis recherchieren (Händler-Inserate 2025/2026)
- Gebrauchtpreise für identische Fahrzeuge recherchieren (min. 5 Referenzen)
- Marktübliche Abzüge für beschriebene Mängel ermitteln

KALKULATION:
1. Basis-Marktwert (Durchschnitt vergleichbarer Fahrzeuge)
2. + Wertsteigerungen (frischer TÜV, Extras, geringe KM)
3. - Wertminderungen (Mängel, Alter, Verschleiß)
4. = Empfohlener Angebotspreis (MIT Verhandlungspuffer)

BITTE ZEIGE MIR DIE KALKULATION ZUR ABNAHME bevor du PDFs erstellst.

DANACH ERSTELLE ZWEI PDFs:
A) INTERN (für mich): Komplette Analyse INKL. Schmerzgrenze
B) KÄUFER (ohne Schmerzgrenze): Transparente Herleitung, verkäufer-optimiert

PDF-STYLE: Professionell, neutral, seriös — nicht wie ein Flyer.
```

## [FIN-02] GEGENSTAND VERKAUFSKALKULATION

**Wann:** Gebrauchsgegenstände (Elektronik, Sport, Haushaltsgeräte) verkaufen.

```
Erstelle Verkaufsunterlagen für folgenden Gebrauchtgegenstand:

OBJEKT: [BESCHREIBEN: Was, Hersteller, Modell, Kaufjahr]
NUTZUNG: [WIE OFT GENUTZT — Stunden / Mal]
ZUSTAND: [Originalkarton / Vollständig / Beschädigungen]
SCHMERZGRENZE (nur für mich): [BETRAG]
PLATTFORM: [eBay Kleinanzeigen / etc.]

AUFGABEN:
1. Aktuellen Neupreis und Gebrauchtmarkt recherchieren
2. Wertminderung kalkulieren (Zeit + Nutzung)
3. Angebotspreis mit Puffer festlegen
4. Kurzes, verkaufsoptimiertes Inserat-Template erstellen

Optimiertes Inserat:
- Headline die Vertrauen schafft
- Zustand ehrlich aber positiv formuliert
- Preis als Anker (kein Verhandlungsraum zeigen)
```
